Aplicaciones Web

La World Wide Web (WWW) o también conocida como “telaraña mundial “es un sistema de intercambio y distribución de información situada en internet.

Esta ha de dejado de ser una inmensa biblioteca de páginas estáticas para convertirse en un servicio que permite acceder a multitud de prestaciones y funciones, así como a la infinidad de servicios, programas, tiendas, información, películas, etc.

La información que está situada en internet podrá acceder miles de usuarios desde diferentes puntos del planeta, por lo cual debe estar perfectamente ordenada y ser fácilmente accesible. Para ello, se utiliza como base el concepto de pagina web.

Una página Web, es un documento electrónico adaptado para la web y que normalmente forma parte de un sitio web. La principal característica son los hipervínculos que ayudan a enlazar con otra página, siendo esto el fundamento de la Web.

La pagina web esta compuesta principalmente por información (texto, módulos, multimedia o ambos) así como por hiperenlaces, además pueden contener o asociar datos de estilo para especificar como se debe visualizar.

Para entender la estructura de una pagina Web, primeramente, deberemos distinguir entre páginas Web estáticas y páginas Web dinámicas. En la página Web estáticas la información aparece siempre tal cual fue escrita y diseñada. En la página Web dinámicas existe códigos de programación que adecuan el contenido de la pagina a las solicitudes de los clientes, normalmente accediendo a una base de datos.
